# acerola3d

Acerola3Dは3Dファイルフォーマットと、これを表示
するためのJava言語の3DCGライブラリです。2003年
ごろに開発を開始して現在はメンテナンスされていません。

大変古い3Dファイルフォーマットですが、2026年に開発を開始した
[a3js](https://github.com/kenji0717/a3js)ライブラリが
サポートする数少ない3Dファイルフォーマットなので、
ライブラリのソースと、サンプルのAcerola3Dファイルを
GitHubで公開することにしました。

* GitHub Pages
    + <https://kenji0717.github.io/acerola3d/>

ライブラリのメンテナンスを再開する予定はありませんが、
中にAcerola3Dフォーマットの規格があります。
Acerola3Dフォーマットのファイルは
[a3js](https://github.com/kenji0717/a3js)ライブラリで
表示させることができます。Acerola3DファイルはGitHub Pagesからも
配信しており
「`https://kenji0717.github.io/acerola3d/A3/ファイル名.a3`」
でCORSアクセスできます。a3jsのプログラミングにてテスト的に
利用していただけます。ただ、本格的に利用される場合は、
ファイルをダウンロードして、ご自分のウェブサーバーなどに
置いて利用して下さい。

サンプルAcerola3Dファイルの作成には、多くの公開された
3Dモデルや音声ファイルを利用させていただいています。
これについては、
[SimpleViewer.html](https://kenji0717.github.io/acerola3d/A3/SimpleViewer.html)
で確認できるようにしております。（だいぶ前に作った物なので、
リング切れも多いのですが・・・）作品を公開してくださって
いる方々に感謝いたします。
